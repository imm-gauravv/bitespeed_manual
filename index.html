Secondary Label: Follow Up

Your digital "Sticky Note".

"I owe this person an action."

The Logic Flow

1
ADD When...

Call no answer. Promise to pay later.

2
ACT

Perform the task (Call back, Reply).

3
REMOVE When...

IMMEDIATELY after task is done.

Rules of Engagement

07 / 16

DO THIS

  • hot-lead + follow-up
  • active-member + follow-up
  • Remove follow-up once resolved.

DON'T DO THIS

  • cold-lead + hot-lead
  • intro-bought + member
  • Leave follow-up on finished task.

System 2: Main Notes

09 / 16

The "Source of Truth"

The permanent record of the member's journey.

Log every call
Log Availability / Holidays
Log Verbal Agreements

Avoid "Ghost Calls"

Scenario: Agent calls, user agrees to buy, agent hangs up but forgets to write a note.

Result:

Next agent sees no history. Calls user again. User gets annoyed. Sale lost.

The C-S-N Formula

10 / 16

Every Main Note must answer 3 questions:

C

Context

What happened?

"Spoke on phone."
S

Status

Situation?

"Price sensitive."
N

Next Step

What to do?

"Call Tues 10am."
Perfect Example

"Spoke today. Wants morning batch. On holiday till Monday. Call Tuesday 10am to book."

The 24-Hour Rule

11 / 16
24h WINDOW

How it works

You can send free-form messages ONLY within 24 hours of the user's last message to us.

When window closes:

You CANNOT message them.
You MUST CALL.

Calling Protocol: "The Double Tap"

12 / 16
1
First Call

Call immediately when a Lead comes in or a Task is due.

2
No Answer? Wait 1 Hour. Call Again.

The "Double Tap". People often ignore the first call.

3
Still No Answer?

1. Add follow-up label.
2. Send WhatsApp (if in 24h window).
